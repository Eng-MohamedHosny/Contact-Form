# Contact Form Project

## Overview
This project is a custom contact form built to practice advanced JavaScript validation logic, form handling, and dynamic UI updates using pure HTML, CSS, and JavaScript.

The form was designed to support scalable validation where adding new fields in HTML automatically integrates with the validation system.

## 🚀 Features

### ✅ Custom Validation System

The form includes a powerful validation engine that dynamically validates required fields.

It handles multiple input scenarios including:

- Required field validation.
- Email format validation.
- Radio group validation.
- Required checkbox validation.

The system is designed to be scalable — meaning new fields can be added in HTML and automatically handled by the validation logic.

---

### ✅ Dynamic Styling & Error Handling

The UI updates dynamically based on validation results:

- Error states are applied automatically to invalid fields.
- Error messages are displayed directly under each field.
- Styling is controlled through JavaScript validation feedback.

This reduces manual handling of UI states.

---

### ✅ Form Group Validation

Special validation was implemented for grouped inputs such as:

- Radio button groups.
- Required checkbox groups.

The logic ensures that at least one option is selected when required.

---

### ✅ Scalable Architecture

The project was designed with maintainability in mind by separating responsibilities into logical layers:

- Input detection layer.
- Validation engine.
- UI state management.
- Error messaging system.

This allows easy extension of the form with new fields.
